## 🛠️ Technology Stack
- **Frontend**: Next.js 14 / 15 (App Router)
- **Styling**: Tailwind CSS (Surgical Utility System)
- **Animations**: Framer Motion (Kinetic Interaction System)
- **Backend**: Next Auth / Supabase / MongoDB
- **UI Components**: Shadcn UI (Custom Studio Fork)

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/workvervenovalko-png/Verve-Nova.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Configure Environment**:
   Create a `.env.local` file with the required enterprise credentials.

4. **Launch Dev Environment**:
   ```bash
   npm run dev
   ```
